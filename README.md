# Remote Studio

Remote Studio is a Linux Mint Cinnamon control layer for RustDesk sessions over Tailscale. It manages headless Xorg display modes, device-specific scaling profiles, a Cinnamon panel applet, and low-latency RustDesk display defaults.

The default `mac` profile targets a 13-inch MacBook Air at `2560x1664`.

Useful commands:

```bash
res
res mac
res doctor
res tailnet
res xorg
```

See [REMOTE_STUDIO.md](REMOTE_STUDIO.md) for install and command details.
